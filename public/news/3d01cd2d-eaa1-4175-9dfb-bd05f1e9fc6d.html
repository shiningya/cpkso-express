30多万的特斯拉Model 3你可能想了解的事情

北京时间2016年4月1日（没错，就是愚人节那天），特斯拉CEO埃隆·马斯克（Elon Musk）在美国加州发布了Model 3车型，该车无论上市前后都备受瞩目。

据悉，该车最快将于 2017 年下半年交付北美市场，2018 年交付中国和亚太地区市场。随着热度持续上升，Model 3在开放公路上测试的频率越来越高，近日网上也再次爆出它的路试谍照图片，下面我们将诸多信息进行汇总，以便您全方位了解它。

亮点提前看：

1、特斯拉Model 3为入门级车型，也是史上最便宜的特斯拉。

2、续航里程可超过345km。

3、全车配备5个座椅，0-100km/h加速时间小于6s，配备自动辅助驾驶硬件，具体自动驾驶级别暂未公布。

4、最终生产将在今年7月份进行进行。

5、根据此前Model 3发布会所显示的信息，Model 3至少将推出5种车身颜色。

Model 3相关问题看这里

1、现在预订 Model 3，什么时候可以提车？

Tesla Model 3 将于2017年末开始投产，将首先在北美地区交付，随后是欧洲和亚太地区；特斯拉会按照实际付款的顺序邀请客户为Model 3选择配置，同时，为感谢现有车主，已拥有一台Model S或Model X的客户将获得优先排产。

2、Model 3 的订金是多少？

每台Model 3的订金是8000 元人民币，每位客户一次最多可以预订2台。

3、Model 3 的售价是多少？

美国地区的基础配置售价约为35,000美元，中国大陆地区的售价待定。

4、什么时候可以试驾 Model 3？

在2017年末正式投产后，将有机会在特斯拉体验店进行实车体验和试驾。

5、Model 3 可以像 Model S 一样享受新能源汽车牌照政策吗？

在签订正式汽车购买协议之前，特斯拉会提供适用于Model 3的新能源汽车激励政策相关信息。

6、如何预订 Tesla Model 3？

Model 3目前仅接受在线预订，通过官网下单并使用支付宝支付订金，提交订单前。

Model 3外观有何亮点？

有消息显示Model 3车辆的尺寸将会较Model S缩小约20%，基本上与奥迪A4的尺寸相仿，从A4的尺寸上来看，长宽高大约分别为4750、1842、1433mm，而现款Model S的尺寸分别为4970、1964、1445mm，尺寸大小基本可以满足使用。

可以肯定的是Model 3依然采用我们熟悉的家族化设计风格，没有太多新意，整体造型看上去很像一台小号Model X，看上去更加紧凑；Model 3的前进气格栅依然会保持全封闭设计，更多外观细节，还得等到量产那天。

车尾有个比较敦实的大屁股，溜背式的造型有利于降低风阻系数，可以更好地节省电力，后备厢上部依然装有上翘的小尾翼，据推测应该为固定的而非电动可调。

今后特斯拉车型都将装备具有充分自动驾驶能力的硬件，这些硬件包括8个摄像头，12个超声波声呐传感器，一个前向雷达以及一台随机携带的新款电脑，其计算能力是之前电脑的40倍。，从路试谍照中也可以看见在前保险杠位置装有4个雷达。

根据现有信息显示：Model 3搭载的Autopilot 2.0系统应该装有 8 个摄像头，其中有 3 个在车辆前脸、2 个在 B 柱、2 个在前挡泥板处，还有一个在车尾。

Model 3所面临的最大挑战是什么？

有消息显示目前Model 3的预订量已经超过40万辆，马斯克原以为每家店可能会有20~30人排队，而不是800人……，这将对特斯拉的产能提出巨大挑战。

去年10月，马斯克宣布由于前12个月的Model 3订单已被抢光，新预定Model 3用户的交付时间将会推迟到2018年中期或者更晚，随后特斯拉发言人再次声明，Model 3交付时间修改，只针对新预定用户，之前预定的用户不会受到任何影响，一开始听到这个消息时内心还是有些崩溃的。

如果特斯拉要全部吃下目前的订单量，其产能就必须要达到2015年的6倍，对此马斯克自己也表示：公司必须慎重制定生产计划，在预计生产力将得到升级的情况下，要完成这些订单大概需要3年时间。所以下订单之后何时才能真正开上它就是另一回事了。

同时，为感谢现有车主，已拥有一台 Model S 或 Model X 的客户将获得优先排产。

Model 3在中国的售价会是多少？

此前，特斯拉发表官方博客正式公布了Model S 85kWh车型的中国区市场零售价为73.4万元，85kWh Performance车型的零售价为85.25万元。Model 3在美国的起售价为35000美元，中国大陆地区的售价待定。因为地区差异、税率不同和运费等因素影响，两地售价不会相同，我们就大胆猜测一下它在中国的售价会是多少？

由于没有排气量所以税特别低，综合关税只有39.5%(关税、增值税等)。35000美元x39.5%+35000美元+运输装卸3600美元= 52425美元，人民币美元汇率6.46x52425=361,208元人民币(约合)。

竞争车型有哪些？

1、纵观纯电动车领域暂时没有直接竞争对手，与其价格接近的只有刚上市不久的北汽EH300和腾势，当这3款车摆在你面前你会选哪个？

2、在这个售价区间内有很多汽油车可与之竞争，例如奔驰C级、宝马3系、奥迪A4L、丰田凯美瑞、本田雅阁、福特蒙迪欧等等，几乎所有中级车都在这一售价区间内，究竟是选择一辆纯电动汽车还是一辆汽油车就要看自己的想法了。

车机互联系统

此前网上也曝光过Model 3内饰细节，对比现在所拍到的图片来看，内饰造型基本可以确定，将与现在在售的特斯拉车型有较大差距，中控台位置的液晶屏很像一块大号iPad，设计非常简洁。




按照马斯克所说那样，内饰将不配备传统汽车上的仪表盘和HUD系统，所有功能将集成在中控大屏幕上。这样设计或许是为了视觉和成本考虑，但从目前情况来看并不能被所有人适应，调整参数需要经过繁琐的步骤和菜单操作，但马斯克给出的说法是未来自动驾驶汽车将成为主流，用户观察仪表盘的频率将会大大降低，但以后的事谁说的准呢？

动力系统如何？


目前对于动力系统没有太多准确消息，特斯拉官网也只是给出了最大续航里程超过 345km、0-100km/h加速时间小于6S和配备超级充电系统这些简单信息。

除官网信息之外，马斯克在今年2月透露，受轴距限制，Model 3 的电池组容量不会超过 100kWh；随后马斯克又说Model 3 电池组最大容量不会超过75kWh；至于驱动形式问题，此前有消息称会采用单电机后驱和双电机四驱两种驱动布局，但发售之初只提供单电机后驱版本。

由于市场定位和售价问题，Model 3整体性能将低于Model S和Model X，因为Model 3肩负着开拓全球市场的重任，目的在于可以让所有人买得起也用得起，也就不需要那样突出和耀眼的性能指标，中庸或许是最好的选项。
